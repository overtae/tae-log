> ✨ **오늘 공부한 것**
>
```
- 모의 면접 문제 풀이
- 영상 상세 페이지 api 연결
- 마이페이지 UI 구현
```
>

- 운영체제의 핵심부, **컴퓨터 자원들을 관리**
- 운영체제는 시스템 프로그램과 커널로 나눌 수 있다.
- **프로그램과 하드웨어 간의 인터페이스 역할**을 담당
    - `하드웨어`  ex. CPU, 메모리, 디스크, 네트워크 장치, 각종 외부 장치 등등 …
    - 사용자 → 시스템 프로그램(ex. Shell) → 커널 → 컴퓨터 자원 접근
